## Sobre o projeto

Projeto criado juntamente com a imersão NLW-POCKET, onde o foco da aplicação é a criação de um sistema de metas, onde o usuario pode adicionar metas durante a semana e definir qual será a frequencia da meta que ele adicionou como objetivo!

Aqui está o porque dessa aplicação:
* Você poderá ter controle e ver em tempo real as suas metas que deverão ser cumpridas, trazendo mais foco para você não se esquecer dos objetivos que você traçou.
* Esse tipo de aplicação pode se extender para diversas features que podem ajudar e melhorar o desempenho que o usuario definir nas metas, trazendo estimulos para continuar cumprindo suas metas.
